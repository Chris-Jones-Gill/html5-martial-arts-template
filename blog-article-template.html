How much will the website design cost?

This is the question we get asked the most, and the one without a simple answer, meaning the answer rarely satisfies.

The problem with the question is because it is too broad, there are so many variables that make up a website design. Without knowing more about the clients design requirements, there is no simple single ammount that is realistic.

The same is true for any design, imagine asking a kitchen designer "How much will a new kitchen cost?". The answer depends on many things, like how many cabinets; how much worktop is needed; what materials do you want; do we need to remove the existing kitchen; and so on.

That being said, we can give guide prices that reflect the type of website being asked about.
